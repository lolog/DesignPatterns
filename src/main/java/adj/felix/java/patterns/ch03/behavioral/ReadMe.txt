03. 行为型模式
    这些设计模式特别关注对象之间的通信。
    包括:
       01) 责任链模式(Chain of Responsibility Pattern)
       02) 命令模式(Command Pattern)
       03) 解释器模式(Interpreter Pattern)
       04) 迭代器模式(Iterator Pattern)
       05) 中介者模式(Mediator Pattern)
       06) 备忘录模式(Memento Pattern)
       06) 观察者模式(Observer Pattern)
       07) 状态模式(State Pattern)
       08) 空对象模式(Null Object Pattern)
       09) 策略模式(Strategy Pattern)
       10) 模板模式(Template Pattern)
       11) 访问者模式(Visitor Pattern)
